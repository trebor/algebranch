# Sidebar button phrasing

Migrated from Antigravity scratch (`button_phrasing_discussion.md`, 2026-06-10).

Question: should the sidebar button that opens the equation input modal say **"Write Equation"** or **"Enter Equation"**?

- "Write": active, creative verb; fits the pen icon. Slightly informal.
- "Enter": standard UI term; feels like data entry.

**Resolution**: with the onboarding work (issue #4), the button was split into a two-button grid: **"Write"** (pen icon, opens input modal) + **"Tutorial"** (book icon, opens onboarding chapter directory). "Write" won.
