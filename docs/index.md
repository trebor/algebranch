# Algebranch

Algebranch is a tool for solving equations by hand. You move terms and apply algebraic identities yourself; the engine only permits valid moves, so the math stays correct at every step. Unlike solvers that hand you the answer, it leaves the decisions to you — which makes sign errors hard to slip in and lets you explore different derivation paths from any point.
